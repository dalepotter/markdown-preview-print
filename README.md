# Markdown to Kindle

A simple markdown editor with Kindle-optimised print view.

## Setup

Install dependencies:

```bash
npm install
```

## Development Scripts

```bash
# Start development server
npm start

# Build for production
npm run build

# Run tests
npm test

# Run tests once (for CI)
npm run test:run
```

## Tech Stack

- **Webpack** - Module bundler
- **Vitest** - Testing framework with JSDOM
- **marked.js** - Markdown parser
